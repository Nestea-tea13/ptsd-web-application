Список опросников


1. Шкала для клинической диагностики ПТСР (Clinical-administered PTSD Scale - CAPS)
Пройти
Еще не прошел месяц с последнего прохождения!
Повторно можно будет пройти опросник через 30 дней после предыдущего заполнения.

Период прохождений: раз в месяц

По результатам прохождения будут получены значения частоты и интенсивности проявления индивидуальных симптомов расстройства.


2. Шкала оценки влияния травматического события (Impact of Event Scale-R - IES-R)
Пройти
Еще не прошла неделя с последнего прохождения!

Период прохождений: раз в неделю

По результатам прохождения будут получены следующие результаты:
  1. Шкала «вторжение» (IN) - позволяет выявить ночные кошмары, навязчивые мысли, связанные с травмирующим событием
  2. Шкала «избегание» (AV) - позволяет выявить симптомы избегания, включающие попытки смягчения или избегания переживаний, связанных с воздействием кризисного события
  3. Шкала «физиологическое возбуждение» (AR) - позволяет выявить злость и раздражительность, гипертрофированную реакцию испуга, психофизиологическое возбуждение, обусловленное воспоминаниями о событии, бессоницу.

3. Шкала Безнадежности Бека (Beck Hopelessness Scale, BHS)
Пройти
Еще не прошла неделя с последнего прохождения!

Период прохождений: раз в неделю

По результатам прохождения будет оценена выраженность негативного отношения по отношению к собственному будущему.


4. Опросник для оценки терапевтической динамики ПТСР «Treatment Outcome PTSD Scale» (TOP-8)
Пройти
Еще не прошла неделя с последнего прохождения!

Период прохождений: раз в неделю

Помогает оценить, насколько мешали симптомы на прошлой неделе.
